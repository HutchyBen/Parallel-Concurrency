### Q1. Rust Hello World

**Question:**

Create a new rust project, and execute the generated hello world code.

**Solution:**

```rs
fn main() {
   println!("Hello world!")
}
```

**Sample output:**

```
PS C:\users\767905\OneDrive - hull.ac.uk\Documents\Parallel Concurrent\Concurrent Labs\lab_a> cargo run
    Finished `dev` profile [unoptimized + debuginfo] target(s) in 0.14s
     Running `target\debug\lab_a.exe`
Hello, world!
```

**Reflection:**
Rust uses Cargo to handle building and packages, to create a new project you run `cargo init` in an empty directory. The directory should follow a snake_case format to follow with Rust naming schemes.

The autogenerated code consists of a main function that calls the println macro.
